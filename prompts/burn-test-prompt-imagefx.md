## NOTES FOR BEST RESULTS

1. **Generate each image separately** — don't try to get all fibers in one prompt
2. **Iterate on the residue images** — these are the most critical for field identification. The difference between a bead (synthetics) and crushable ash (wool) must be unmistakable
3. **The polyester smoke MUST look dramatically heavier** than the others — this is the #1 field identifier for polyester
4. **Wool must look different from all synthetics** — it self-extinguishes (no sustained flame), produces ash not beads, and smells like hair
5. **Consistent lighting and style** across all images so they look like they belong in the same training manual
6. **If using ImageFX**: Try adding "photorealistic" or "DSLR photography" to the prompt for more realistic results
7. **If using Midjourney**: Add `--ar 1:1 --s 250 --q 2` for square format, high stylization, and max quality
