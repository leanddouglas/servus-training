## HOW TO USE THESE PROMPTS

1. Open [Whisk](https://labs.google/whisk)
2. For each prompt below, paste the **Subject**, **Scene**, and **Style** into the matching Whisk fields
3. Upload the **Servus logo** (navy-on-white version OR white-on-navy version) as a **Style** reference image for brand consistency
4. You can also upload your existing ImageFX burn test photos as the **Subject** reference to keep visual consistency

## TIPS FOR WHISK

1. **Upload the Servus logo as a Style reference image** — use the white-on-navy version for dark backgrounds, navy-on-white for light backgrounds. This helps Whisk match the brand colors
2. **Upload your ImageFX photos as Subject reference** — this keeps the visual content consistent across all generated images and videos
3. **For videos**: Start with the flame prompts (1A, 2A, 3A, 4A) — flame motion is the most impactful for student memory
4. **Best video prompts**: 6B (polyester black smoke) and 6C (olefin dripping) create the most dramatic, memorable motion
5. **For the comparison shots** (Section 5): Generate as images first, then try video mode for an animated "reveal" effect
6. **Subject field is most important** — be ultra-specific about what the fiber looks like when burning. Scene and Style refine the mood and branding
7. **Iterate**: If the flame colors aren't right, add more specific color descriptions to the Subject field (e.g., "Pantone 2728 C blue base transitioning to Pantone 151 C orange tip")
8. **Brand consistency**: Every output should include either the servus logo watermark OR the navy branded bar. If Whisk isn't rendering the logo text correctly, generate the content without it and add the Servus branding as an overlay in post-production
